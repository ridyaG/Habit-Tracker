## Story 1: View Personal Information

### Title:
As a user, I want to view my saved name, username, age, and country on my profile page, so that I can see the details I provided during registration.

### Acceptance Criteria:

1. Profile page displays stored user info.

2. Fields are non-editable by default.

3. Edit button enables update mode.

* Priority: Medium
* Story Points: 2

## Story 2: Edit Personal Information

### Title:
As a user, I want to update my name, username, age, and country, so that I can keep my information up to date.

### Acceptance Criteria:

1. Editable fields become active on clicking edit.

2. Changes validated before saving.

3. Cancel restores previous values.

* Priority: High
* Story Points: 3

## Story 3: Save Updated Information

### Title:
As a user, I want the changes I make to my profile to be saved, so that my updated details are stored and reflected throughout the app.

### Acceptance Criteria:

1. Clicking save updates user info.

2. Success message shown after save.

3. Updated info visible across all pages.

* Priority: High
* Story Points: 3

## Story 4: Update Name in Header

### Title:
As a user, I want my updated name to be displayed in the app's header after I change it in the profile, so that my changes are immediately visible.

### Acceptance Criteria:

1. Header updates immediately with new name.

2. No reload required.

3. Persisted across all screens.

* Priority: Medium
* Story Points: 1
