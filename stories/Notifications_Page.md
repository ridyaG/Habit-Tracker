## Story 1: Enable / Disable Notifications

### Title:
As a user, I want to be able to enable or disable notifications, so that I can choose whether or not to receive reminders.

### Acceptance Criteria:

1. Toggle switch controls app-wide notifications.

2. Current setting saved persistently.

3. System confirms change visually.

* Priority: High
* Story Points: 2

## Story 2: Add Habits for Notifications

### Title:
As a user, I want to select specific habits to receive notifications for, so that I only get reminders for the habits I am actively working on.

### Acceptance Criteria:

1. List of habits includes a checkbox for notifications.

2. Selected habits trigger alerts as scheduled.

3. Changes saved automatically.

* Priority: Medium
* Story Points: 2

## Story 3: Set Notification Times

### Title:
As a user, I want to receive notifications three times a day (morning, afternoon, evening), so that I get timely reminders throughout the day.

### Acceptance Criteria:

1. User can select time slots (morning/afternoon/evening).

2. Notifications trigger at chosen times.

3. System handles time zone correctly.

* Priority: High
* Story Points: 3
